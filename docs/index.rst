Installation
------------

To install the Movieclassifier Python project,
run this command in your terminal:

.. code-block:: console

   $ pip install movieclassifier


Usage
-----

Movieclassifier Python's usage looks like:

.. code-block:: console

   $ movieclassifier [OPTIONS]

.. option:: -t <train>, --train <train>

    Training model

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
